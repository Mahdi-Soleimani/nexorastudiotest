# Nexora AI Studio Website

وب‌سایت چندصفحه‌ای برای معرفی استودیوی Nexora AI، شامل صفحات خانه، راهکارها، نمونه‌کارها، آکادمی، تیم، فرصت‌های شغلی و تماس است. طراحی با تم تاریک و رنگ بنفش نئونی انجام شده و یک چت‌بات شناور به عنوان دمو ایجنت در همه صفحات حضور دارد.

## ساختار پروژه
- `index.html` – صفحه خانه با معرفی کوتاه، شاخص‌ها و ماشین‌حساب ROI.
- `services.html` – توضیح سرویس‌های اتوماسیون و ایجنت‌ها به همراه FAQ.
- `portfolio.html` – نمونه‌کارها با فیلتر دسته‌بندی.
- `academy.html` – ویدیوها، نکته‌ها و فرم خبرنامه.
- `team.html` – داستان تیم، ارزش‌ها و کارت اعضا.
- `careers.html` – فرهنگ سازمانی، موقعیت‌های باز و فرم ارسال رزومه.
- `contact.html` – فرم B2B و اطلاعات تماس.
- `assets/css/style.css` – استایل اصلی.
- `assets/js/main.js` – منطق تعامل (فعال‌سازی منو، فیلتر پروژه، FAQ، ROI و چت شناور).

## اجرای محلی
از هر وب‌سرور ساده می‌توانید استفاده کنید. مثال با پایتون:

```bash
python -m http.server 8000
```

سپس در مرورگر آدرس `http://localhost:8000` را باز کنید.
