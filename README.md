# Nexora AI Studio Website

یک وب‌سایت چندصفحه‌ای برای معرفی استودیوی «Nexora AI Studio» شامل صفحات خانه، راهکارها، نمونه‌کارها، آکادمی، تیم، فرصت‌های شغلی و تماس است. طراحی با تم تاریک نئونی انجام شده و المان‌هایی مانند چت‌بات شناور، فیلتر نمونه‌کارها و ماشین‌حساب ROI تعاملی دارد.

## ساختار
- `index.html`: صفحه خانه با معرفی سریع، آمار و ماشین‌حساب ROI کوچک.
- `services.html`: توضیح خدمات، جدول مقایسه و ROI کامل.
- `portfolio.html`: کارت پروژه‌ها با فیلتر دسته‌بندی.
- `academy.html`: ویدیوهای یوتیوب، مقالات و فرم خبرنامه.
- `team.html`: داستان تیم و کارت اعضا با مهارت‌ها.
- `careers.html`: فرهنگ تیم، موقعیت‌های شغلی و فرم درخواست.
- `contact.html`: فرم B2B، اطلاعات تماس و چت‌بات.
- `assets/css/style.css`: استایل اصلی با تم بنفش/آبی نئونی.
- `assets/js/main.js`: منطق ناوبری فعال، فیلتر پروژه، ROI و چت‌بات.

## استفاده
صفحه `index.html` را در مرورگر باز کنید یا کل دایرکتوری را روی GitHub Pages مستقر کنید. تمام صفحات به شکل استاتیک بوده و به هم لینک شده‌اند.
